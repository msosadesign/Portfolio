Instant Pet Finder Signup Form

Instant Pet Finder is an idea for an app that focuses on finding cute pets for cute users. Iâ€™ve been given the task to make the registration section on a wireframe. So I used design patters to make an enjoyable and engaging flow.

* The options show up while the user is selecting, this helps him/her decide with a tinier number of options and therefore having to think less. Showing all the options at the same time could overwhelm the user.

This project was interesting to me because I learned about Design Patterns, how people are used to certain repeating aspects of design, and how we can help our users implementing this in our favor. I learned to apply these patterns, how to inform users about their current actions, how to organize information and overall, how to make a design less frustrating.
